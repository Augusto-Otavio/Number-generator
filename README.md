<h1 align="center">Number Generator</h1>

<p align="center">
  <strong>Aplicação para sorteio de números aleatórios, com opções de intervalo, quantidade e sorteio sem repetição.</strong>
</p>

<p align="center">
  Este projeto foi desenvolvido como parte do aprendizado em JavaScript,
  com foco em lógica de programação, manipulação do DOM, validação de dados
  e experiência do usuário.
</p>

<h2>Funcionalidades</h2>

<h3>Configuração do Sorteio</h3>
<ul>
  <li>Definição do número mínimo</li>
  <li>Definição do número máximo</li>
  <li>Escolha da quantidade de números a serem sorteados</li>
  <li>Opção para gerar números sem repetição</li>
</ul>

<h3>Geração de Resultados</h3>
<ul>
  <li>Sorteio de números aleatórios dentro do intervalo definido</li>
  <li>Geração com ou sem repetição</li>
  <li>Exibição visual dos números sorteados</li>
  <li>Animação sequencial para cada número gerado</li>
</ul>

<h3>Controle de Interface</h3>
<ul>
  <li>Ocultação do formulário após o sorteio</li>
  <li>Exibição do botão "Sortear novamente" ao final da animação</li>
  <li>Reset completo da interface para um novo sorteio</li>
</ul>

<hr>

<h2>Interface e Experiência do Usuário</h2>
<ul>
  <li>Layout simples e intuitivo</li>
  <li>Feedback visual claro durante o sorteio</li>
  <li>Animações suaves e sequenciais</li>
  <li>Foco na clareza e facilidade de uso</li>
</ul>

<hr>

<h2>Tecnologias Utilizadas</h2>

<h3>HTML5</h3>
<ul>
  <li>Estrutura semântica da aplicação</li>
  <li>Uso correto de formulários e inputs numéricos</li>
  <li>Organização clara dos elementos da interface</li>
</ul>

<h3>CSS3</h3>
<ul>
  <li>Layout responsivo utilizando Flexbox e Grid</li>
  <li>Animações com <code>@keyframes</code></li>
  <li>Uso de <code>animation-delay</code> para efeito sequencial</li>
  <li>Controle de visibilidade com classes utilitárias</li>
</ul>

<h3>JavaScript</h3>
<ul>
  <li>Manipulação do DOM</li>
  <li>Validação de entradas do usuário</li>
  <li>Geração de números aleatórios</li>
  <li>Lógica para sorteio sem repetição</li>
  <li>Uso de <code>Number()</code> e <code>Number.isNaN()</code> para validações seguras</li>
  <li>Controle de tempo com <code>setTimeout</code></li>
  <li>Organização do código em funções reutilizáveis</li>
</ul>
